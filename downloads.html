Menuen til venstre indeholder linkst til en række filer. Python og Pygame er nødvendige for at kunne deltage i forløbet. Python er den fortolker, der kører de programmer, som I kommer til at skrive. Installation

Python

Python installeres ved at downloade den første fil og køre den. Før man "klikker igennem" de forskellige skridt i installeren, skal man sætte et "flueben" ved "Add Python 3.5 to PATH".

Pygame

Pygame er noget værre at installere. Men rolig, vi skal nok hjælpe. Proceduren er:

  • Åben et konsol vindue. Gøres ved at trykke på "windows knapper", taste "cmd" i søgefeltet og trykke enter. Der skulle gerne komme et sort vindue med grå tekst og en cursor, hvor man kan indstate kommandoer, frem.
  • I dette vindue tastes kommandoen "pip install wheel".
  • Download Pygame
  • Dernæst tastes kommandoen "pip install {stien til pygame filen}". Hvor teksten mellem { og } skal ertattes med stien til filen. Denne kan evt. indtastes ved at hive fil-ikonet ned i konsol-vinduet.

Installationen kan testes ved at starte programmet Idle, der er en brugergrænseflade til Python. I hvide vindue der kommer, skriver man så "import pygame" efter de tre ">" tegn og trykker enter. Hvis den blot svarer med tre nye ">" tegn er alt gået og Pygame er korrekt installeret.
